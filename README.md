<h4 align="center">支持多家云存储的公私兼备的云服务系统.</h4>

本来此项目并没有这么快开始，但由于我曾经使用过的某个类似产品的作者吃相实在难看，故决定自己实现一个。

此项目的愿景是集百家之长（Cloudreve + Alist + FnOS）。你也可以考虑付费支持我们的发展 -> `DiskNext Pro`.

目前正处于 `OMEGA` 实验阶段，比 `Alpha` 版还更早期，仅供测试。
